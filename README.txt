How to use:
1. Download and unzip the folder.
2. Open index.html in Chrome (Android) or a modern desktop browser.
3. Grant microphone permission when the browser asks.
4. Use the mic button or type commands in the input box.

Commands:
- "YouTube <term>"
- "Mausam" or "Weather"
- "Samay" or "Time"
- "Khabar" or "News"
- "Mazak" or "Joke"
- "Wikipedia <term>"
- "Band karo" or "exit"

To make it installable (PWA) on Android:
- Open the site in Chrome, then use Chrome menu -> Add to Home screen.

To host online (free):
- Create a GitHub repository and upload the files.
- In repository settings enable GitHub Pages from the main branch.
- Or use Netlify / Vercel drag-and-drop deploy.
